The Ganymede XML Client

The Ganymede server and client suites include a text-mode xmlclient, which can be used to load data and schema information into the Ganymede server, and which may be used to dump data from the server to an XML file.

Usage:


    xmlclient file.xml

        This will cause the xmlclient to connect to the Ganymede server
        and to submit the file.xml for processing.

    xmlclient -dumpschema

	This will cause the xmlclient to connect to the Ganymede server
	and to dump a <ganymede> xml file to stdout that includes a
	<ganyschema> section defining the current schema definition in
	the Ganymede server.

    xmlclient -dumpdata

	This will cause the xmlclient to connect to the Ganymede server
	and to generate a complete dump of the server's data objects
	to stdout in xml format.

    xmlclient -dump

        This will cause the xmlclient to connect to the Ganymede server
	and to generate a complete dump of the server's database,
	both schema and data, to stdout in xml format.

    Optional parameters:

        username=<persona name>

        password=<password>

      Examples:

        xmlclient username=jonhart:supergash -dumpschema > schema.xml

        xmlclient username=broccol password=notReal changes.xml
